Date Picker

I think the only challenge here would be in styling the default input and maybe for bonus points create 2 from scratch in the javascript portion of my journey. Here we have 5 different styles for a native date picker.
Aside: I really lucked out because in researching this mini-project I saw that the native date picker was not supported in safari until 2021 :)
